    • The DOM is used to access the document and its elements using JavaScript. Without the DOM, Javascript wouldn't have any notion on what is going on in the webpage. The DOM is not apart of the Javascript language but is instead a web API used to build websites. The Dom was designed to be independent of any particular programming language making the structural representation of the document available from a single, consistent API.
